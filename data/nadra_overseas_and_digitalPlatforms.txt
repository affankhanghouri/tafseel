--------------------------------------------------------------------------------
[TOPIC: PAK IDENTITY FOREIGN COUNTERS — STEP-BY-STEP PROCESS]
--------------------------------------------------------------------------------
NADRA has counters inside 22 Pakistani Missions worldwide for NICOP, POC, and other services.

STEP 1 : Schedule appointment at the Pakistani Mission's website BEFORE visiting.
STEP 2 : Bring all required documents to the Pak Identity counter manager.
         If documents are complete, proceed to next step.
STEP 3 : Select processing category: Normal / Urgent / Executive (different fees apply).
STEP 4 : Photograph is captured live at the counter.
STEP 5 : Pay via credit card / debit card (Visa/Mastercard).
STEP 6 : Fingerprints captured using thumb machine.
STEP 7 : Counter manager completes the form; a system-generated verifier form printed,
         signed by applicant, scanned, and uploaded.
STEP 8 : Supporting documents scanned and uploaded to system.
STEP 9 : Application submitted.

NOTE: If additional documents required, NADRA contacts you by email or phone.

--------------------------------------------------------------------------------
[TOPIC: APPLYING FOR NICOP WHILE VISITING PAKISTAN]
--------------------------------------------------------------------------------
Overseas Pakistanis visiting Pakistan can apply at any NRC for NICOP.
Fees charged in Pakistani Rupees (PKR):
- Normal (31 days): Rs. 5,500
- Urgent (23 days): Rs. 8,300
- Executive (9 days): Rs. 11,000
This is often convenient for dual nationals visiting family who want to update records in person.

[TOPIC: PASSPORT APPLICATION FOR OVERSEAS PAKISTANIS]
--------------------------------------------------------------------------------
Overseas Pakistanis can renew passport via:
1. Online portal: https://onlinemrp.dgip.gov.pk
2. Visit nearest Pakistani Embassy/Mission with MRP facility.

ONLINE (Overseas) available categories:
- Renewal
- New (for newborns up to 2 years)
- Reprint/Damaged/Lost/Exhaust
- Modification (female surname, marital status, profession change)

HOME DELIVERY available for overseas applicants (unless restricted by local law, e.g., Iran, Morocco).

IMPORTANT: For ANY data modification on passport (name, DOB, etc.):
- Must FIRST modify CNIC/NICOP at NADRA.
- THEN apply for passport with revised NADRA card.

--------------------------------------------------------------------------------
[TOPIC: DIGITAL ID — HOW IT WORKS]
--------------------------------------------------------------------------------
DEFINITION: Legal, secure, verifiable digital credentials issued by NADRA under the NADRA Ordinance 2000.

REGISTRATION REQUIREMENTS:
- Must be 18 years or older
- A SIM card registered on your CNIC
- Biometric verification (photo + fingerprints via Pak ID app)

HOW TO GET DIGITAL ID:
1. Create Pak ID account and complete biometric verification.
2. Digital ID is stored in Pak ID Vault automatically upon verification.
3. Use Digital ID to access government, health, education, and financial services.

DIGITAL ID COMPONENTS:
- Identity Number: Valid CNIC/NICOP/POC + biometric verification
- Mobile Number: Verified via OTP (Pakistani mobile numbers only; international numbers not PTA-verified)
- Email Address: Verified via OTP
- Biometrics: Fingerprints OR facial recognition

TECHNICAL STANDARDS COMPLIANCE:
- OAuth 2.0 and OpenID Connect (OIDC) — for secure authentication
- NIST 800-63 — multi-factor authentication (MFA) + biometrics
- TLS 1.2/1.3 — secure data transmission
- JWT (JSON Web Token) — secure identity claims
- ISO/IEC 27001 — real-time monitoring, fraud prevention
- ISO/IEC 24760 — eKYC, biometric verification, document validation
- GDPR/Pakistan Data Protection Bill — explicit user consent for data sharing
- Pakistan Security Standards — pre-quantum cryptographic controls

--------------------------------------------------------------------------------
[TOPIC: SSO — SINGLE SIGN-ON ("Sign In With PAKID")]
--------------------------------------------------------------------------------
DESCRIPTION: One login credential to access multiple government and private sector portals.
Eliminates the need for separate usernames/passwords for each service.

ELIGIBLE USERS:
- Must have a NADRA-issued Citizen Identification Number (CNIC/NICOP/POC)
- Must complete biometric verification on Pak ID platform
- Mobile number verified via OTP
- Email verified via OTP
- Fingerprints verified against NADRA records
- Facial verification against NADRA records

WHAT SSO RETURNS TO ORGANIZATIONS:
(Only these 5 verified attributes — nothing more, nothing less)
- First Name
- Last Name
- Verified CNIC Number
- Verified Mobile Number
- Verified Email Address

NOTE: SSO is NOT a data-sharing service. It only authenticates the user.
      Organizations needing additional citizen data must use NADRA's eKYC / MyInfo service.

PLATFORM SUPPORT: Web and mobile (iOS SDK and Android SDK available for developers).
STANDARDS: OIDC / OAuth 2.0 compliant.

BENEFITS FOR CITIZENS:
- One login for all government services (passport, tax, health, education)
- No repeated registrations
- Protection against phishing and identity theft
- Consent-based data sharing; audit logs of where your identity was used
- Works for both inland and overseas Pakistanis

--------------------------------------------------------------------------------
[TOPIC: E-PASSPORT — OVERVIEW]
--------------------------------------------------------------------------------
LAUNCHED: 2022 (soft launch with Diplomatic and Official categories)
NATIONWIDE: Available at all field offices since August 16, 2023.

FEATURES:
- RFID Chip (embeds biometric data)
- PKI (Public Key Infrastructure)
- Integration with biometric features (Facial + Fingerprint)
- IPI (Invisible Personal ID)
- 2D Barcode
- Machine Readable Zone (MRZ)
- ICAO compliant (international travel standard)

E-PASSPORT FEES (5-year validity, includes service charges):
36 pages — PKR based (higher than standard MRP; check current DGIP rates)

DIFFERENCE — MRP vs e-Passport:
- MRP: Standard machine-readable, cost-effective, widely recognized.
- e-Passport: Higher security, biometric chip, faster processing at border control.
- Fast Track service is NOT available for e-Passport delivery.

FINGERPRINT VERIFICATION FOR E-PASSPORT: Can now be done via NADRA PAKID Mobile App.

[SECTION: SUCCESSION CERTIFICATE — DETAILED PROCESS]
================================================================================

INTRODUCED: January 2021 — NADRA replaces courts for succession certificate issuance.
IMPACT: Reduced judiciary workload by 30%.
AVAILABLE IN: ICT, Punjab, Sindh, Khyber Pakhtunkhwa. Expanding to other regions.

--------------------------------------------------------------------------------
[TOPIC: HOW TO APPLY FOR SUCCESSION CERTIFICATE]
--------------------------------------------------------------------------------
APPLICATION POINT: District-level NADRA office. Must be initiated by a legal heir IN PAKISTAN.

REQUIRED DOCUMENTS:
1. Applicant's CNIC/NICOP
2. Deceased person's Death Certificate
3. Deceased person's CNIC/NICOP
4. Letter of authorization from ALL legal heirs, authorizing one person to file the application.

FEES:
- Assets > Rs. 100,000: Rs. 20,000 (Rs. 22,000 in Sindh)
- Assets < Rs. 100,000: Rs. 10,000

PROCESS: 5-step simplified process. No court visit required.

FOR OVERSEAS APPLICANTS: Application must be initiated in Pakistan at a NADRA centre.
Overseas heirs must provide Letter of Authorization (Power of Attorney) to a person in Pakistan.
